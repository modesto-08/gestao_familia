## 💻 Sobre o Projeto

O **FamilyHub** é um sistema web completo desenvolvido para gerenciar a rotina real de uma família. O objetivo é centralizar a organização de membros e atividades em uma interface intuitiva e responsiva.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** do **SENAI**, aplicando conhecimentos integrados de Front-end, Back-end e Banco de Dados.

---

## ⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento, foram aplicadas as seguintes funcionalidades:

**Gestão de Membros:** Cadastro completo dos integrantes da família (Pai, Mãe, Filho, etc.).
**Gestão de Atividades:** Controle de compromissos escolares, esportivos e sociais.
**Visualização em Calendário:** Interface gráfica para acompanhamento temporal das tarefas.
**Notificações:** Sistema de alertas para compromissos agendados.
**Arquitetura de Dados:** - Relacionamento **1:N** (Um membro para várias atividades).
Uso de Chaves Primárias e Estrangeiras para integridade dos dados.
**Interface Moderna (UI/UX):** - Menu lateral de navegação.
Dashboard com cards de status rápido.
